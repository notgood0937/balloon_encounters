## Adding a New Panel

<Callout type="warn">
Registering a new panel requires synchronized changes across **6 files**. Missing any step will cause runtime errors or invisible panels.
</Callout>

<Steps>
  <Step>
    ### Update Store

    `src/stores/uiStore.ts` — Add to `DEFAULT_PANEL_VISIBILITY`, `DEFAULT_PANEL_ORDER`, and the `hydrateFromPrefs` migration logic.
  </Step>
  <Step>
    ### Update Settings Modal

    `src/components/SettingsModal.tsx` — Add to the `PanelVisibility` interface and `PANEL_LABELS` map.
  </Step>
  <Step>
    ### Update Preferences

    `src/hooks/usePreferences.ts` — Add default value in `DEFAULT_PREFERENCES` and include in `panelOrder`.
  </Step>
  <Step>
    ### Register in Page

    `src/app/page.tsx` — Import the component, add to `DEFAULT_COL_SPANS`, `panelHandlers` ids array, and `renderPanel` switch case.
  </Step>
</Steps>

## Layout System

| Zone | Max Columns | Typical Usage |
|------|------------|---------------|
| **Right sidebar** | 2 | Market Detail, Smart Money, Portfolio |
| **Bottom bar** | 3 | Charts, Orderbook, Whale Trades, Signals |

Panels are identified by `data-panel` DOM attributes for drag-and-drop sorting. Column spans are configurable per-panel via `DEFAULT_COL_SPANS`.
